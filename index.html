Project Samples

  • PizzaBot

    Chatbot interface powered by Artificial Intelligence

    A personal project to learn more about the world of A.I. and challenge myself to code a highly-polished user-interface with transitions and animations using the latest ES6 version of React.

    • Chatbot interface built from the ground up without the use of CSS frameworks or additional libraries other than pure React.js
    • Integration with DialogFlow's API; Google's own Artificial Intelligence platform (formerly api.ai)
    • Fully responsive for the best experience on both web and mobile browsers
    • Use of React's latest ES6 version
    • Deployed to Netlify for maximum performance

    Tech build summary: HTML5, CSS3, React.js, DialogFlow API, Netlify

  • Drumpfsy

    An Trump-themed Etsy shop.

    An ecommerce "shell" that uses the Etsy API to show hand-made goods inspired by the current Republican Presidential candidate President of the United States: Mr. Donald J. Trump.

    • React.js for building the site's user interface
    • Backbone.js used for creating home/single-page routes and and creating data models
    • Fetching Etsy API data for displaying humorous current markeplace items related to our dear President

    Tech build summary: HTML5, Sass, React.js, Backbone.js, Etsy API, Heroku

  • Blue Leaf

    Ecommerce website with gift registry software integration.

    Blue Leaf is a trendy lifestyle store in the heart of Houston. After being in business for a few years, they decided to jump into the ecommerce space and start selling their goods online.

    • UPS & USPS shipping carrier integration for live-pricing quotes
    • Integration with Registry Valet, a third party registry software
    • Use of the Wordpress CMS for easy product catalog maintenance, page updates, and customer order management
    • Training different members of the Blue Leaf sales team to use the new website and manage & ship customer orders
    • Mobile-responsiveness for optimal user experience

    Tech build summary: HTML5, CSS3, Wordpress, WooCommerce, Registry Valet integration with custom PHP coding.
